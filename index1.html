Use Cases for Event Page

  • I want to find out when Stone Temple Pilots is playing on Saturday.
  • I want to find out how much a 3 day pass is.
  • I want to learn about The Smashing Pumpkins.
  • I want to learn about the event.

    • Steps

    Step 1. Navigate through the page with Control-Option-Right Arrow.

    Step 2. Once you get to Artist link select it with Control-Option-Space bar.

    Step 3. Open list of items on the page with Control-Option-I.

    Step 4. Go through the list until you come across The Smashing pumpkins.
